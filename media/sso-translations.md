### Problem karty (3 karty)

| KEY | EN — title | EN — desc | CS title | CS desc |
|-----|-----------|----------|---------|---------|
| problem.1.title | Fragmented Identity | Fans create separate accounts for every touchpoint: website, mobile app, fanshop, kiosk. Nothing is connected, nothing is shared. | | |
| problem.2.title | Lost Fan Data | Without a single profile, there is no personalization, no cross-selling, and no way to understand what a fan actually does across your channels. | | |
| problem.3.title | Security Gaps | Multiple password stores, inconsistent security policies, no central audit trail. Every separate system is a separate liability. | | |
